# Setup My Linux

## Usage

### Clone Repository

```bash
$ git clone git@github.com:zekhoi/setup-my-linux.git
```

### Run Script

```bash
$ ./setup.sh [options]
```
## Options

| Option | Description |
| --- | --- |
| latest | Install latest version of all ubuntu packages |
| node | Install nodejs |
| go | Install golang |
| php | Install php |
| python | Install python |
| docker | Install docker |
| dev | Install development tools |
